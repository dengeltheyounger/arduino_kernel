Set up the stack for a given task.

The complexity is that the address of do_task needs to be added to the stack. The reason is that the housekeeper will then pop the address of do_task from the task's stack when the task is selected.

Parameters
tThe task for which the stack needs to be set up.
task_numThe total number of tasks.
sThe stack that the tasks will use.
stack_numThe total number of stacks to be parceled.

\ret 0 if there are more stacks than tasks, 1 if otherwise.
